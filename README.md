# Pinnacle_labs_intern_sentiment_analysis_webapp
#  Sentiment Analysis Web Application

## 📌 Project Overview

This project is a Machine Learning-based **Sentiment Analysis Web Application** that predicts the sentiment of user-provided text as **Positive** or **Negative**. The application is built using **Python**, **Flask**, and **Scikit-learn**, with **TF-IDF Vectorization** and a **Logistic Regression** model for text classification.

The project performs text preprocessing by removing URLs, punctuation, numbers, stop words, and converting text to lowercase before transforming the text into numerical features using TF-IDF. The trained model is then used to predict the sentiment through a simple and interactive web interface.



## 🚀 Features

* Predicts sentiment of user-entered text.
* Text preprocessing using Natural Language Processing (NLP).
* TF-IDF Vectorization for feature extraction.
* Logistic Regression classification model.
* Interactive Flask-based web application.
* Clean and responsive user interface using HTML & CSS.
* Model serialization using Joblib.
* Easy to run and customize.

## ⚙️ Machine Learning Workflow

1. Load the dataset.
2. Perform text preprocessing.
3. Remove stop words and unwanted characters.
4. Convert text into TF-IDF vectors.
5. Train the Logistic Regression model.
6. Save the trained model and vectorizer.
7. Build a Flask web application.
8. Predict sentiment from user input.



## 📊 Dataset

The project uses the **Sentiment140** dataset containing Twitter posts labeled with sentiment classes.

* Text preprocessing
* Feature extraction using TF-IDF
* Sentiment prediction using Logistic Regression
